# Assets Folder

This folder contains all static assets for the portfolio website.

## Required Files

- `favicon.ico` - Website favicon (replace with your custom favicon)
- `resume.pdf` - Your resume in PDF format (to be added)

## Optional Folders

- `projects/` - Project screenshots and images
  - `ems.jpg` - EMS project image (example)

## How to Add Assets

1. **Favicon**: 
   - Create a 32x32 or 16x16 icon
   - Convert to .ico format
   - Replace `favicon.ico`

2. **Resume PDF**:
   - Export your resume as PDF
   - Save as `resume.pdf` in this folder

3. **Project Images**:
   - Create `projects/` subfolder
   - Add project screenshots
   - Update image paths in `js/projects.js`
